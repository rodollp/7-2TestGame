### 경험치 및 골드 시스템
- 골드는 자동 획득, 경험치는 직접 획득하는 구조로 분리
- MonsterDrop에서 몬스터 사망 이벤트(OnDead)를 통해 경험치 오브와 골드 오브를 생성하도록 구현
- 경험치 오브는 플레이어의 수집 범위 안에 들어왔을 때만 추적을 시작하도록 제작
- 골드 오브는 생성 즉시 플레이어를 추적하여 자동으로 획득되도록 제작
- PlayerWallet을 추가하여 골드 획득 및 관리를 Status와 분리

### 무기 데이터 구조 개선
- WeaponData를 레벨별 능력치를 관리하는 구조로 변경
- WeaponLevelData를 추가하여 각 레벨마다 데미지, 쿨타임, 사거리 등의 능력치를 개별 설정할 수 있도록 구현
- WeaponType을 근접(Melee), 투사체(Projectile), 범위(Area)로 재정리하여 공격 방식을 명확하게 구분

### 무기 인벤토리
- WeaponStatus를 MonoBehaviour가 아닌 일반 C# 클래스로 변경
- WeaponInventory를 제작하여 최대 3개의 무기를 보관할 수 있도록 구현
- 동일한 무기를 획득하면 새로운 슬롯을 사용하는 대신 레벨이 증가하도록 구현
- 빈 슬롯이 있을 경우 새로운 WeaponStatus를 생성하여 인벤토리에 추가하도록 구현

### 공격 시스템 수정
- AutoAttack이 WeaponStatus의 현재 레벨 데이터를 참조하도록 구조 변경
- PlayerAttack에서 플레이어 공격력과 무기 공격력을 합산하여 최종 데미지를 적용하도록 수정
