# 2026-07-15 Dev Log

## 오늘 구현한 내용

### Monster Spawn System 설계

- StageData를 이용한 시간 기반 몬스터 스폰 구조 설계
- SpawnData를 이용하여 시간 구간별 스폰 규칙 관리
- MonsterSpawnManager 구현
- MonsterSpawner 구현

---

### SpawnData 구조

SpawnData에는 다음 정보를 저장하도록 설계하였다.

- StartTime
- EndTime
- SpawnInterval
- SpawnCount
- MonsterPrefabs

이를 이용하여 시간 구간마다 다른 몬스터를 생성하도록 구성하였다.

---

### MonsterSpawner 구현

MonsterSpawner의 역할

- SpawnData를 전달받는다.
- 생성할 몬스터를 랜덤 선택한다.
- 플레이어 주변 랜덤 위치를 계산한다.
- NavMesh 위의 위치를 찾는다.
- 몬스터를 생성한다.

---

### MonsterSpawnManager 구현

MonsterSpawnManager의 역할

- 게임 시간 관리
- SpawnData 변경
- SpawnInterval 계산
- MonsterSpawner에게 생성 요청

---

### 오류 수정

- MonsterStatus 클래스 이름 변경으로 인해 MonsterData 연결이 해제된 문제 수정
- StageDuration이 0초로 설정되어 스폰이 동작하지 않던 문제 수정
